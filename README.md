# Описание проекта

**Цель проекта — построить надёжную модель для предсказания цен поддержанных японских автомобилей.**

### Подход включает:

Очистку данных и обработку пропусков

**Feature engineering:** возраст авто, квадрат возраста, логарифм возраста, отношение мощности к весу, взаимодействия признаков

Кодирование категориальных признаков (включая признаки с высокой кардинальностью)

Разделение данных на обучающую, валидационную и тестовую выборки *(70/15/15)*

Пайплайны препроцессинга для числовых и категориальных признаков

**Ансамбль** из 5 нейронных сетей, объединённых усреднением предсказаний

Оценка качества с помощью **MSE, RMSE, MAE, MAPE, R²**

Визуализация распределения ошибок и сопоставления предсказанных и реальных цен

### Результаты: Ансамбль достигает высокой точности: R² ≈ 0.9976, MAPE ≈ 1.42%, что демонстрирует почти идеальное предсказание цен автомобилей.
